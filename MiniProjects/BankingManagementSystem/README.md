# 🏦 Banking Management System v1.0.0

A console-based Banking Management System developed in C. This project provides essential banking operations such as account management, deposits, withdrawals, money transfers, transaction history, interest calculation, and report generation using file handling.

---

## 📌 Features

* Create New Account
* Deposit Money
* Withdraw Money
* Transfer Money Between Accounts
* Check Account Balance
* Search Account by Number
* Search Account by Name
* Update Account Details
* Delete Account
* Display All Accounts
* Secure PIN Verification
* Transaction History
* Interest Calculator
* Generate Account Statement
* Export Account Report
* File-Based Data Storage
